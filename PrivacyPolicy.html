隐私政策将描述Zr如何收集和使用您提供的信息。

一、所有使用的权限

我们会在收集和使用您的存储信息之前向您询问，如果您不希望使用您的存储权限，请在系统设置选项中禁用。

当应用导入.OPML类文件及下载图片时，我们需要请求授予存储权限：

  • android.permission.READ_EXTERNAL_STORAGE
  • android.permission.WRITE_EXTERNAL_STORAGE

当应用需要访问Rss源时，我们需要获取网络权限：

  • android.permission.INTERNET

我们使用的信息：

  • 无

三、卸载应用

  • 当您卸载APP时，所有数据将会自动清空。

四、安全

  • 我们没有连接任何远程服务，应用服务功能在本地完成所有操作，不存在网络传输数据，安全性有保证，可放心使用。

The Privacy Policy will describe how Zr collects and uses the information you provide.

1. All usage permissions

We will ask you before collecting and using your storage information. If you do not want to use your storage permissions, please disable it in the system settings options.

When the application imports the .OPML class file, we need to request storage permission:

  • android.permission.READ_EXTERNAL_STORAGE
  • android.permission.WRITE_EXTERNAL_STORAGE

When the application needs to access the Rss source, we need to obtain network permissions:

  • android.permission.INTERNET

2. Collection and use of information

Information we collect:

  • None

Information we use:

  • None

3. Uninstall the application

  • When you uninstall the APP, all data will be cleared automatically.

4. Safety

  • We are not connected to any remote services. The application service function completes all operations locally. There is no network transmission of data. The security is guaranteed and can be used with confidence.
